Nosso projeto

O projeto Truck-in consiste em uma API desenvolvida usando Java spring-boot e um website desenvolvido utilizando HTML, CSS e JavaScript. Com essas ferramentas desenvolveremos um sistema aque irá auxiliar a jornada trabalhista do motorista de caminhão durante suas viagens e possibilitar o empregador ter uma visão mais ampla de onde está o motorista, que horas ele irá fazer suas pausas, qual motorista dirige qual caminhão, entre outras coisas.

Como funciona

É simples! Você poderá controlar as frotas e acompanhar todo o percurso dos caminhoneiros colaboradores da empresa .

Serviços

Será disponibilizado um web service onde terá todas as informações do veículo e caminhoneiro. E também funções para cadastrar os mesmos!

Truck-In

O Truck-in irá auxiliar os caminhoneiros, pois com o software ele irá conseguir estabelecer as suas paradas e onde o motorista se encontra durante sua jornada de trabalho.
